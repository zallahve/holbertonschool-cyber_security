# cyber_prompt_injection

This project is about prompt injection attacks against LLM-based systems.

## Learning objectives

- Understand what prompt injection and jailbreaks are
- Learn the main types of prompt injection
- Understand how attackers build injection payloads
- Learn basic mitigation techniques
- Practice safe and ethical red-teaming in a sandboxed lab

For this project, the main target is:

## Notes
