# 0x05. Wireshark basics

This directory contains answers for **Wireshark display filter** tasks.

## Files

- `0-ip_scan.txt`  
  Wireshark filter to detect packets generated by an **Nmap IP protocol scan**:
  `nmap -sO <target>`

## Notes

- Filters are **Wireshark display filters** (not capture filters).
- The goal is to isolate traffic patterns produced by specific scan techniques.
