# 0x03 Cryptography Basics

## Description
Basic cryptography tasks (hashing, cracking tools, and related fundamentals) for Holberton Cyber.

## Files
- `0-sha1.sh`: Hashes a provided password using sha1 and writes the digest to `0_hash.txt`.
- `1-sha256.sh`: Hashes a provided password using sha256 and writes the digest to `1_hash.txt`.
